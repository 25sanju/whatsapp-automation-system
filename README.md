# WhatsApp Automation System

A Python GUI-based WhatsApp automation tool built using Tkinter, Pandas, and PyWhatKit.

## Features
- Upload CSV contacts
- Send bulk WhatsApp messages
- GUI-based interface
- Automated message delivery

## Technologies Used
- Python
- Tkinter
- Pandas
- PyWhatKit

## How to Run
1. Install required libraries:
   python -m pip install pandas pywhatkit

2. Run:
   python gui.py

3. Upload a CSV file containing contacts.

## Sample CSV Format

name,phone
User1,+919876543210
User2,+919123456789
User3,+919998887777
